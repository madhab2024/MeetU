## ğŸ“¦ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- MongoDB (Local or Atlas connection string)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/madhab2024/MeetU.git
   cd G_MEET
   ```

2. **Setup Backend**
   ```bash
   cd Backend
   npm install
   ```
   Create a `.env` file in the `Backend` directory and configure your environment variables (e.g., `MONGO_URL`).

3. **Setup Frontend**
   ```bash
   cd ../Frontend
   npm install
   ```

### Running the Application

1. **Start the Backend**
   ```bash
   # In the Backend directory
   npm run dev
   ```

2. **Start the Frontend**
   ```bash
   # In the Frontend directory
   npm run dev
   ```
